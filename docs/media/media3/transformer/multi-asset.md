Using Transformer, you can combine multiple media assets, such as videos,
images, and audio files into a single video. To get started, learn about
[arranging your input media items into a `Composition`](https://developer.android.com/media/media3/transformer/composition), Media3's
representation for multi-asset editing projects. You can then
[preview your project with `CompositionPlayer`](https://developer.android.com/media/media3/transformer/compositionplayer), or [export](https://developer.android.com/media/media3/transformer/getting-started#start) the
`Composition` with the `Transformer` API.
