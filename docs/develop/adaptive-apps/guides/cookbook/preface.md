# Adaptive app cookbook Stay organized with collections Save and categorize content based on your preferences.





Android provides all the ingredients for five-star adaptive apps. The recipes in
this cookbook select and combine choice ingredients to solve specific
development problems. Each recipe includes best practices, quality code samples,
and step-by-step instructions to help you become an adaptive app top chef.

## Star ratings

The recipes are star rated based on how well they align with the [Adaptive app
quality guidelines](/docs/quality-guidelines/adaptive-app-quality).

|  |  |
| --- | --- |
| Five star rating icon | Meets the criteria for Tier 1, Adaptive differentiated |
| Four star rating icon | Meets the criteria for Tier 2, Adaptive optimized |
| Three star rating icon | Meets the criteria for Tier 3, Adaptive ready |
| Two star rating icon | Provides some adaptive capabilities, but falls short of the adaptive app quality guidelines |
| One star rating icon | Meets the needs of a specific use case, but doesn't properly support adaptive app behavior |
