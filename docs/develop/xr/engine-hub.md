### [Direct Preview](http://developer.android.com/develop/xr/engine-hub/direct-preview)

Direct Preview lets you test and iterate on complex interactions directly inside your game engine's editor using live data from the Android XR Device. This completely bypasses the usual, time-consuming build-and-deploy cycle within XR development. [Set up Direct Preview](http://developer.android.com/develop/xr/engine-hub/direct-preview)
