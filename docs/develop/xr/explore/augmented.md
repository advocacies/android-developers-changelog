### [Build UI with Jetpack Compose Glimmer](http://developer.android.com/develop/xr/jetpack-xr-sdk/jetpack-compose-glimmer)

Jetpack Compose Glimmer is a UI toolkit for building augmented Android XR experiences, optimized for display glasses. Build beautiful, minimal, and comfortable UI for devices that are worn all day. [Build UI with Jetpack Compose Glimmer](http://developer.android.com/develop/xr/jetpack-xr-sdk/jetpack-compose-glimmer) [Learn about Jetpack Compose Glimmer components design](http://developer.android.com/design/ui/ai-glasses/guides/components/overview) [![](http://developer.android.com/static/images/develop/xr/jetpack-xr-sdk/conversational.png)](http://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live)

### [Build conversational experiences](http://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live)

Integrate AI and contextual awareness into your app's experiences for AI glasses. Build conversational experiences with the Gemini Live API that enable voice to voice interaction with Gemini models. [Integrate with the Gemini Live API](http://developer.android.com/develop/xr/jetpack-xr-sdk/gemini-live) [Learn about designing audio flow \& queues](http://developer.android.com/design/ui/ai-glasses/guides/foundations/get-started#audio-flow-cues) ![](http://developer.android.com/static/images/spot-icons/jetpack.svg)

## Supported tools and technologies

The Jetpack XR SDK includes all the tools you need to build augmented experiences for AI glasses.  

Use Jetpack Compose Glimmer, Android Studio, the emulator, and ARCore for Jetpack XR. [Get started](http://developer.android.com/develop/xr/jetpack-xr-sdk/get-started)
