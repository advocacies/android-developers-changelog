# AEP guideline: Material UX Stay organized with collections Save and categorize content based on your preferences.





Use Material Design 3, Android's recommended design system, to build beautiful,
usable, accessible, and consistent products.

## Required implementation

To qualify for AEP, apps must use the [Compose Material 3](/jetpack/androidx/releases/compose-material3) library or the
[Wear Compose Material 3](/jetpack/androidx/releases/wear-compose-m3) library for Wear OS devices.

## Guideline applicability

This guideline applies:

* To apps that support a comparable UX implementation on a non-Android
  platform.
* To all form factors on which the app is available.

## Exemptions

There are no exemptions for this guideline.
