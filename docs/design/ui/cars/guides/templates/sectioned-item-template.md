> [!NOTE]
> **Note:** With the introduction of the Sectioned Item template, [list](https://developer.android.com/design/ui/cars/guides/templates/list-template) and [grid](https://developer.android.com/design/ui/cars/guides/templates/grid-template) templates are no longer needed. To get the latest updates and features, move any existing instances of the list or grid templates into the Sectioned Item template.

Sectioned Item template is a vertically scrolling layout that supports mixed
component groups. You can arrange rows and grids sequentially to create a
customized browsing structure.

Sections can include:

- Lists
- Grids
- Chip
- Banner
- Condensed items
- Spotlight sections

> [!NOTE]
> **Note:** In Sectioned Item template, you can use Expanded Headers to support rich layouts.

![Image representing a sectioned item](https://developer.android.com/static/images/design/ui/cars/templates/media-sectioned-item.png) This sample shows a list and a grid used together in the Sectioned Item template.

## Sectioned item template UX requirements

To make sure your lists and grids adhere to UX best practices, see
[List template](https://developer.android.com/design/ui/cars/guides/templates/list-template) and [Grid template](https://developer.android.com/design/ui/cars/guides/templates/grid-template).
