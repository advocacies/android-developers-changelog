> [!NOTE]
> **Note:** Including the minimized control panel is required to meet [car app quality guidelines](https://developer.android.com/docs/quality-guidelines/car-app-quality) for all media apps using Car App Library version 1.9 or higher.

![hero](https://developer.android.com/static/design/ui/cars/guides/components/minimized-control-panel/minimized-control-panel-assets/image-161-21370.png)

## Overview

The minimized control panel is a persistent, responsive, condensed version of
the Now Playing View (NPV) interface that remains visible throughout the user's
browsing journey.

For media apps built with the Car App Library (CAL), the minimized control panel
appears during active playback sessions to provide uninterrupted access to core
controls.

*** ** * ** ***

## Composition

Designed for high glanceability, the panel dynamically adapts its component
layout based on available screen width.

Minimized control panels for media consist of 3 parts:

- **Image and texts:** Title and subtexts should be restricted to 1 line
- **Primary actions**
- **Progress bar**
