> [!NOTE]
> **Note:** Start building with Car App Library version 1.9 components to create more differentiated and expressive in-car experiences.

The [1.9.0-alpha01](https://developer.android.com/jetpack/androidx/releases/car-app#version_19_2) release of the Car App Library (CAL) lets you build
more differentiated, expressive experiences across Android Auto and cars with
Google built-in. This page provides guidance on what each component is and how
to apply it.

## New components (1.9)

- [Spotlight section](https://developer.android.com/design/ui/cars/guides/components/spotlight-section/spotlight-section): Showcase featured content collections or editorial recommendations.
- [Condensed items](https://developer.android.com/design/ui/cars/guides/components/condensed-items/condensed-items): Display compact visual components to fit more content on the screen.
- [Chip](https://developer.android.com/design/ui/cars/guides/components/chips/chips): Allow users to filter content or trigger quick actions.
- [Minimized control panel](https://developer.android.com/design/ui/cars/guides/components/minimized-control-panel/minimized-control-panel): Persistent, condensed controls for active media playback.
- [Banner](https://developer.android.com/design/ui/cars/guides/components/banners/banners): Showcase special app experiences, new features, or important messages.
- [Section header](https://developer.android.com/design/ui/cars/guides/components/section-header/section-header): Label and provide controls for individual content groups.

## Existing components

- [Action strip](https://developer.android.com/design/ui/cars/guides/components/action-strip): Put secondary or tertiary actions a tap away.

- [Map action strip](https://developer.android.com/design/ui/cars/guides/components/map-action-strip): Give users access to map interactivity features.

- [Button](https://developer.android.com/design/ui/cars/guides/components/button): Provide access to actions that drivers may need to take.

- [Floating action button (FAB)](https://developer.android.com/design/ui/cars/guides/components/fab): Place the most important actions on the
  screen a tap away.

- [Header](https://developer.android.com/design/ui/cars/guides/components/header): Help drivers understand which app and what part of the app
  they're using.

- [Row](https://developer.android.com/design/ui/cars/guides/components/row): Display a small amount of text, plus other options.
