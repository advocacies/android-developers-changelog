Expand your game beyond mobile phones to reach users on tablets, foldables,
ChromeOS devices, car displays, TVs, and XR. With Compose APIs, you can build
games that automatically adjust their layouts and controls for any screen size,
shape, or orientation.

Enhance your game's accessibility and provide a premium experience on every
supported device.

## Adaptive apps development hub

Go to the [Adaptive apps development hub](https://developer.android.com/develop/adaptive-apps/guides/games/build-adaptive-apps-for-games) to learn about building adaptive
games, including how to:

- Create adaptive game layouts
- Modify your game's navigation based on window size classes
- Use window size classes for dynamic UI adjustments
- Utilize canonical layouts such as list‑detail with Jetpack APIs
